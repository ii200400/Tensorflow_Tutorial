# 텐서보드와 모델 재사용

## 모델을 저장하고 불러오기

 + 체크포인트 파일(checkpoint file) : 특정 시점에 모델 변수의 상태를 포착한 데이터를 의미한다. \
 단, 그래프 자체는 체크포인트에 포함되지 않는다.
